![[MCS1_HW001 תרגיל בית 1 2025-11-03 21.08.11.excalidraw.svg]]^figure-actuator-series
>קבל קבוע מחובר בטור למשפעל מסוג קבל לוחות מקבילים.


### סעיף א'
רשמו את האנרגיה הפוטנציאלית הכוללת של המערכת עבור שתי הקונפיגורציות (מקבילי וטורי).

**פתרון**:
עבור קבלים מחוברים במקביל, אנו יודעים ש[[PHY2_003 אלקטרוסטטיקה - מוליכים, אנרגיה אלקטרוסטטית, קבלים#חיבור קבלים במקביל|ניתן פשוט לחבר את הקיבול]] ולהתייחס אליהם כקבל יחיד שקול:

$$C'_{p}=C+{C}_{0}$$
בנוסף, אנו מכירים מההרצאה ([[MCS1_001 מבוא#הפעלת מטען|משוואה]] $\text{(1.8)}$) ש- $C= \dfrac{{\varepsilon}_{0}A}{g-x}$ הוא הקיבול של הקבל המשתנה, ולכן:
$$C'_{p}=\dfrac{{\varepsilon}_{0}A}{g-x}+{C}_{0} \tag{HW1.1}$$

מנקודת המבט האלקטרוסטטית:

$$Q=C'_{p}V$$

נסכום את האנרגיות הפוטנציאליות במערכת:

$$\begin{aligned}
{\psi}_{p} & =\dfrac{1}{2}kx^{2}+\dfrac{1}{2} \dfrac{Q^{2}}{C'_{p}}+({Q}_{\text{battery}}-Q)V \\[1ex]
 & =\dfrac{1}{2}kx^{2}+\dfrac{1}{2}{C}'_{p}V^{2}+{Q}_{\text{battery,0}}V-C'_{p}V^{2} \\[1ex]
 & =\dfrac{1}{2}kx^{2}-\dfrac{1}{2}C'_{p}V^{2}+{Q}_{\text{battery,0}}V
\end{aligned}$$

נציב את $\text{(HW1.1)}$:
$$\boxed {
{\psi}_{p}=\dfrac{1}{2}kx^{2}-\dfrac{1}{2}\left( \dfrac{{\varepsilon}_{0}A}{g-x} +{C}_{0}\right)V^{2}+{Q}_{\text{battery,0}}V
 }$$

במקרה הטורי, הקיבול החדש יהיה:
$$\begin{gathered}
\dfrac{1}{C'_{s}}=\dfrac{1}{C}+\dfrac{1}{{C}_{0}} \\[1ex]
C'_{s}=\dfrac{C {C}_{0}}{C+{C}_{0}}
\end{gathered}$$
נציב את [[MCS1_001 מבוא#הפעלת מטען|משוואה]] $\text{(1.8)}$:
$$\begin{aligned}
C'_{s} & = \left( \dfrac{{\varepsilon}_{0}A}{g-x} \right) \dfrac{{C}_{0}}{[{\varepsilon}_{0}A/(g-x)]+{C}_{0}} \\[1ex]
 & =\dfrac{{\varepsilon}_{0}A{C}_{0}}{{\varepsilon}_{0}A+{C}_{0}(g-x)}
\end{aligned}$$
לכן סך האנרגיה הפוטנציאלית היא כעת:
$$\boxed {
{\psi}_{s}=\dfrac{1}{2}kx^{2}-\dfrac{1}{2}\left( \dfrac{{\varepsilon}_{0}A{C}_{0}}{{\varepsilon}_{0}A+{C}_{0}(g-x)} \right)V^{2} +{Q}_{\text{battery,0}}V
 }$$

### סעיף ב'
מצאו את הכוח הריאקטיבי (גזרו את האנרגיה לפי דרגת החופש).

**פתרון**:
עבור החיבור במקביל:
$$\boxed {
{f}_{R,p}=kx-\dfrac{1}{2} \dfrac{{\varepsilon}_{0}A}{(g-x)^{2}}V^{2}
 }$$
 נשים לב שהוא זהה לחלוטין לכוח הריאקטיבי של [[MCS1_001 מבוא#הפעלת מתח|המשפעל בהפעלת מתח מההרצאה]]!

 עבור החיבור הטורי:
 $$\boxed {
{f}_{R,s}=kx-\dfrac{1}{2} \left( \dfrac{{\varepsilon}_{0}A{{{C}_{0}}}^{2}}{[{\varepsilon}_{0}A+{C}_{0}(g-x)]^{2}} \right)V^{2}
 }$$

### סעיף ג'
רשמו את תנאי שיווי המשקל.

**פתרון**:
נקבל שיווי משקל כאשר ${f}_{R}=0$. עבור החיבור במקביל:
$$\begin{gathered}
\boxed {
kx=\dfrac{1}{2} \dfrac{{\varepsilon}_{0}A}{(g-x)^{2}}V^{2}
 }
\end{gathered}$$

עבור החיבור הטורי:
$$\boxed{kx=\dfrac{1}{2} \left( \dfrac{{\varepsilon}_{0}A{{{C}_{0}}}^{2}}{[{\varepsilon}_{0}A+{C}_{0}(g-x)]^{2}} \right)V^{2} }$$

### סעיף ד'
מצאו את קשיחות המערכת.

**פתרון**:
נקבל את קשיחות המערכת לאחר גזירת הכוח הריאקטיבי בדרגת החופש. עבור החיבור במקביל:
$$\boxed{{K}_{p}=k- \dfrac{{\varepsilon}_{0}A}{(g-x)^{3}}V^{2} }$$

עבור החיבור הטורי:
$$\boxed{{K}_{s}=k-\left( \dfrac{{\varepsilon}_{0}A{{{C}_{0}}}^{3}}{[{\varepsilon}_{0}A+{C}_{0}(g-x)]^{3}} \right)V^{2} }$$

### סעיף ה'
מהי הקשיחות במצב ש"מ?

**פתרון**:
נציב את תנאי השיווי משקל. עבור החיבור המקבילי נוכל לרשום את התנאי כ:
$$\begin{gathered}
kx=\dfrac{1}{2} \dfrac{{\varepsilon}_{0}A}{(g-x)^{2}}V^{2} \\[1ex]
\dfrac{{\varepsilon}_{0}A}{(g-x)^{3}}V^{2}=\dfrac{2kx}{(g-x)}
\end{gathered}$$
נציב בקשיחות:
$$\begin{gathered}
{K}_{\text{eq},p}=k-\dfrac{2kx}{(g-x)} \\[1ex]
\boxed{{K}_{\text{eq},p}=k\left( \dfrac{g-3x}{g-x} \right) }
\end{gathered}$$

עבור החיבור הטורי, התנאי:
$$\begin{gathered}
kx=\dfrac{1}{2} \left( \dfrac{{\varepsilon}_{0}A{{{C}_{0}}}^{2}}{[{\varepsilon}_{0}A+{C}_{0}(g-x)]^{2}} \right)V^{2} \\[1ex]
\left( \dfrac{{\varepsilon}_{0}A{{{C}_{0}}}^{3}}{[{\varepsilon}_{0}A+{C}_{0}(g-x)]^{3}} \right)V^{2}=\dfrac{2kx{C}_{0}}{{\varepsilon}_{0}A+{C}_{0}(g-x)}
\end{gathered}$$
נציב בקשיחות:
$$\begin{gathered}
{K}_{\text{eq},s}  =k-\dfrac{2kx{C}_{0}}{{\varepsilon}_{0}A+{C}_{0}(g-x)} \\[1ex]
\boxed {
{K}_{\text{eq},s} =k\left( \dfrac{{\varepsilon}_{0}A+{C}_{0}g-3{C}_{0}x}{{\varepsilon}_{0}A+{C}_{0}(g-x)} \right)
 }
\end{gathered}$$

### סעיף ו'
האם יש ערך קריטי בו המערכת מאבדת יציבות? אם כן מה הוא?

**פתרון**:
עבור החיבור המקבילי, נקבל את אותם התוצאות מההרצאה ([[MCS1_001 מבוא#הפעלת מתח|משוואות]] $\text{(1.9)}$-$\text{(1.10)}$):

$$\boxed {
\begin{aligned}
 & {x}_{\text{PI}}=g/3 \\[1ex]
 & {V}_{\text{PI}}=\sqrt{ \dfrac{8}{27} \dfrac{kg^{3}}{{\varepsilon}_{0}A} }
\end{aligned}
 }$$
כאשר $V>{V}_{\text{PI}}$ אנו מאבדים יציבות.

עבור החיבור הטורי, נמצא מתי ${K}_{\text{eq},s}=0$:
$$\begin{gathered}
{K}_{\text{eq},s}=0 \\[1ex]
{\varepsilon}_{0}A+{C}_{0}g-3{C}_{0}x=0 \\[1ex]
\boxed {
{x}_{\text{PI}}=\dfrac{{\varepsilon}_{0}A+{C}_{0}g}{3{C}_{0}}
 }
\end{gathered}$$
נציב את ${x}_{\text{PI}}$ בתנאי שיווי המשקל כדי למצוא את ${V}_{\text{PI}}$. ראשית נשים לב ש:
$$\begin{aligned}
{\varepsilon}_{0}A+{C}_{0}(g-{x}_{\text{PI}}) & ={\varepsilon}_{0}A+{C}_{0}g-{C}_{0}{x}_{\text{PI}} \\[1ex]
 & ={\varepsilon}_{0}A+{C}_{0}g-\dfrac{{\varepsilon}_{0}A+{C}_{0}g}{3} \\[1ex]
 & =\dfrac{2({\varepsilon}_{0}A+{C}_{0}g)}{3}
\end{aligned}$$
נציב בתנאי ש"מ:
$$\begin{gathered}
k{x}_{\text{PI}}  =\dfrac{1}{2} \left( \dfrac{{\varepsilon}_{0}A{C}_{0}^{2}}{[{\varepsilon}_{0}A+{C}_{0}(g-{x}_{\text{PI}})]^{2}} \right){V}_{\text{PI}}^{2} \\[1ex]
k\cdot\dfrac{{\varepsilon}_{0}A+{C}_{0}g}{3{C}_{0}} =\dfrac{1}{2} \left( \dfrac{{\varepsilon}_{0}A{C}_{0}^{2}}{\left[\dfrac{2({\varepsilon}_{0}A+{C}_{0}g)}{3}\right]^{2}} \right){V}_{\text{PI}}^{2} \\[1ex]
k\cdot\dfrac{{\varepsilon}_{0}A+{C}_{0}g}{3{C}_{0}} =\dfrac{1}{2} \left( \dfrac{9{\varepsilon}_{0}A{C}_{0}^{2}}{4({\varepsilon}_{0}A+{C}_{0}g)^{2}} \right){V}_{\text{PI}}^{2}
\end{gathered}$$
נפתור עבור ${V}_{\text{PI}}$:
$$\boxed {
{V}_{\text{PI}}=\sqrt{ \dfrac{8k({\varepsilon}_{0}A+{C}_{0}g)^{3}}{27{\varepsilon}_{0}A{{{C}_{0}}}^{3}} }
 }$$
כאשר $V>{V}_{\text{PI}}$ אנו מאבדים יציבות.

### סעיף ז'
בחנו את הביטוי לערך הקריטי שקיבלתם ונסו להסיק האם ישנו ערך סופי של קיבול קבוע בו המערכת מגיבה בצורה מיוחדת? ישנו יחס מיוחד בין הקיבול ההתחלתי של הקבל המשתנה (כאשר $x=0$) והקבל הקבוע. האם ישנו יחס קיבול מסוים בו המערכת מגיבה בצורה מיוחדת?

**פתרון**:
עבור החיבור הטורי, נשים לב שנקבל ${x}_{\text{PI}}<g$ (כלומר נקודת pull-in שהיא בכלל פיזיקלית) רק אם:
$$\begin{gathered}
\dfrac{{\varepsilon}_{0}A+{C}_{0}g}{3{C}_{0}}<g \\[1ex]
{\varepsilon}_{0}A<2{C}_{0}g \\[1ex]
{C}_{0}> \dfrac{1}{2}\dfrac{{\varepsilon}_{0}A}{g}
\end{gathered}$$
בנוסף, נזכור כי ${C}_{i}=\dfrac{{\varepsilon}_{0}A}{g}$ כאשר ${C}_{i}$ הוא הקיבול של הקבל המשתנה כאשר $x=0$. נסיק שאם ${C}_{0}\leq \dfrac{1}{2}{C}_{i}$, לא יהיה pull-in למערכת.

נביט שוב בביטוי ל-${x}_{\text{PI}}$ עבור החיבור הטורי:
$${x}_{\text{PI}}=\dfrac{{\varepsilon}_{0}A+{C}_{0}g}{3{C}_{0}}$$
כאשר ${C}_{0}=\dfrac{1}{2} \dfrac{{\varepsilon}_{0}A}{g}$ ראינו ש- ${x}_{\text{PI}}=g$. מעבר לגודל זה, ${x}_{\text{PI}}$ מתרחק מ-$x=g$ עד שב- ${C}_{0}\to \infty$, אנו מגיעים בחזרה ל- ${x}_{\text{PI}}=g/3$.

## שאלה 2 - הפעלת מטען

### סעיף א'
רשמו את האנרגיה הפוטנציאלית הכוללת של המערכת עבור שתי הקונפיגורציות (מקבילי וטורי).

**פתרון**:
הראנו כבר כי עבור קבלים מחוברים במקביל:

$$C'_{p}=C+{C}_{0}$$
מההרצאה, ([[MCS1_001 מבוא#הפעלת מטען|משוואה]] $\text{(1.8)}$) אנו יודעים ש- $C= \dfrac{{\varepsilon}_{0}A}{g-x}$ הוא הקיבול של הקבל משתנה, ולכן:
$$\begin{align}
C'_{p} & =\dfrac{{\varepsilon}_{0}A}{g-x}+{C}_{0} \tag{HW1.1} \\[1ex]
 & = \dfrac{{\varepsilon}_{0}A+{C}_{0}(g-x)}{(g-x)}
\end{align}$$
נסכום את האנרגיות הפוטנציאליות במערכת:
$$\begin{aligned}
{\psi}_{p} & =\dfrac{1}{2}kx^{2}+\dfrac{1}{2} \dfrac{Q^{2}}{C'_{p}}
\end{aligned}$$
נציב את $\text{(HW1.1)}$:
$$\boxed {
{\psi}_{p}=\dfrac{1}{2}kx^{2}+\dfrac{1}{2} \dfrac{(g-x)}{{\varepsilon}_{0}A+{C}_{0}(g-x)}Q^{2}
 }$$

במקרה הטורי, הקיבול החדש יהיה:
$$\begin{gathered}
\dfrac{1}{C'_{s}}=\dfrac{1}{C}+\dfrac{1}{{C}_{0}} \\[1ex]
C'_{s}=\dfrac{C {C}_{0}}{C+{C}_{0}}
\end{gathered}$$
נציב את [[MCS1_001 מבוא#הפעלת מטען|משוואה]] $\text{(1.8)}$:
$$\begin{aligned}
C'_{s} & = \left( \dfrac{{\varepsilon}_{0}A}{g-x} \right) \dfrac{{C}_{0}}{[{\varepsilon}_{0}A/(g-x)]+{C}_{0}} \\[1ex]
 & =\dfrac{{\varepsilon}_{0}A{C}_{0}}{{\varepsilon}_{0}A+{C}_{0}(g-x)}
\end{aligned}$$
לכן סך האנרגיה הפוטנציאלית היא כעת:
$$\boxed {
{\psi}_{s}=\dfrac{1}{2}kx^{2}+\dfrac{1}{2} \dfrac{{\varepsilon}_{0}A+{C}_{0}(g-x)}{{\varepsilon}_{0}A{C}_{0}} Q^{2}
 }$$
### סעיף ב'
מצאו את הכוח הריאקטיבי (גזרו את האנרגיה לפי דרגת החופש).

**פתרון**:
עבור החיבור במקביל:
$$
\begin{aligned}
{f}_{R,p} & =kx+\dfrac{1}{2} Q^{2} \dfrac{(-1)[{\varepsilon}_{0}A+{C}_{0}(g-x)]-(g-x)(-{C}_{0})}{[{\varepsilon}_{0}A+{C}_{0}(g-x)]^{2}} \\[1ex]
 & =kx+\dfrac{1}{2}Q^{2} \dfrac{-{\varepsilon}_{0}A-{C}_{0}(g-x)+{C}_{0}(g-x)}{[{\varepsilon}_{0}A+{C}_{0}(g-x)]^{2}}
\end{aligned}
$$
לכן:
$$\boxed{{f}_{R,p}=kx-\dfrac{1}{2} \dfrac{{\varepsilon}_{0}A}{[{\varepsilon}_{0}A+{C}_{0}(g-x)]^{2}}Q^{2} }$$

עבור החיבור בטור:
$$\boxed {
{f}_{R,s}=kx-\dfrac{1}{2} \dfrac{Q^{2}}{{\varepsilon}_{0}A}
 }$$
נשים לב שהוא זהה לחלוטין לכוח הריאקטיבי של [[MCS1_001 מבוא#הפעלת מטען|המשפעל בהפעלת מטען מההרצאה]]!

### סעיף ג'
רשמו את תנאי שיווי המשקל.

**פתרון**:
נקבל שיווי משקל כאשר ${f}_{R}=0$. עבור החיבור במקביל:
$$\begin{gathered}
\boxed {
kx=\dfrac{1}{2} \dfrac{{\varepsilon}_{0}A}{[{\varepsilon}_{0}A+{C}_{0}(g-x)]^{2}}Q^{2}
 }
\end{gathered}$$

עבור החיבור הטורי:
$$\boxed{kx=\dfrac{1}{2} \dfrac{Q^{2}}{{\varepsilon}_{0}A} }$$

### סעיף ד'
מצאו את קשיחות המערכת.

**פתרון**:
נקבל את קשיחות המערכת לאחר גזירת הכוח הריאקטיבי בדרגת החופש. עבור החיבור במקביל:
$$\boxed{{K}_{p}=k-\dfrac{{\varepsilon}_{0}A{{{C}_{0}}}}{[{\varepsilon}_{0}A+{C}_{0}(g-x)]^{3}} Q^{2}}$$

עבור החיבור הטורי:
$$\boxed{{K}_{s}=k }$$


### סעיף ה'
מהי הקשיחות במצב ש"מ?

**פתרון**:
נציב את תנאי השיווי משקל. עבור החיבור המקבילי נוכל לרשום את התנאי כ:
$$\begin{gathered}
kx=\dfrac{1}{2} \dfrac{{\varepsilon}_{0}A}{[{\varepsilon}_{0}A+{C}_{0}(g-x)]^{2}}Q^{2} \\[1ex]
\dfrac{{\varepsilon}_{0}A}{[{\varepsilon}_{0}A+{C}_{0}(g-x)]^{3}}Q^{2}= \dfrac{2kx}{[{\varepsilon}_{0}A+{C}_{0}(g-x)]}
\end{gathered}$$
נציב בקשיחות:
$$\begin{gathered}
{K}_{\text{eq},p}=k-\dfrac{2kx{{{C}_{0}}}}{{{\varepsilon}_{0}A+{C}_{0}(g-x)}} \\[1ex]
\boxed{{K}_{\text{eq},p}=k\left( \dfrac{{\varepsilon}_{0}A+{C}_{0}g-3{C}_{0}x}{{\varepsilon}_{0}A+{C}_{0}(g-x)} \right) }
\end{gathered}$$
שזוהי תוצאה זהה לחלוטין לקשיחות בש"מ בשאלה הקודמת עבור חיבור בטור (אבל עם הפעלת מתח).

עבור החיבור הטורי, הקשיחות קבועה ללא קשר לדרגת החופש, ולכן:
$$\boxed{{K}_{\text{eq},s}=k }$$

### סעיף ו'
האם יש ערך קריטי בו המערכת מאבדת יציבות? אם כן מה הוא?

**פתרון**:
עבור החיבור המקבילי, נשים לב שהחישוב מאוד דומה ל[[#שאלה 1 - הפעלת מתח#סעיף ו'|סעיף ו' בשאלה הקודמת]], מאחר וקיבלנו קשיחות זהה (עבור הפעלת מתח, קבל מחובר בטור). נמצא מתי ${K}_{\text{eq},s}=0$:
$$\begin{gathered}
{K}_{\text{eq},s}=0 \\[1ex]
{\varepsilon}_{0}A+{C}_{0}g-3{C}_{0}x=0 \\[1ex]
\boxed {
{x}_{\text{PI}}=\dfrac{{\varepsilon}_{0}A+{C}_{0}g}{3{C}_{0}}
 }
\end{gathered}$$
נציב את ${x}_{\text{PI}}$ בתנאי שיווי המשקל כדי למצוא את ${Q}_{\text{PI}}$. מאחר והחישוב דומה מאוד ל[[#שאלה 1 - הפעלת מתח#סעיף ו'|סעיף ו' בשאלה הקודמת]]:
$$\boxed {
{Q}_{\text{PI}}=\sqrt{ \dfrac{8k({\varepsilon}_{0}A+{C}_{0}g)^{3}}{27{\varepsilon}_{0}A{C}_{0}} }
 }$$
כאשר $V>{V}_{\text{PI}}$ אנו מאבדים יציבות.

עבור החיבור הטורי, אין ערך קריטי בו אנו מאבדים יציבות מאחר ו- $K>0$ לכל $x$.

### סעיף ז'
עבור החיבור המקבילי, נקבל את אותם התוצאות כמו ב[[#שאלה 1 - הפעלת מתח#סעיף ז'|סעיף ז' בשאלה קודמת]] עבור חיבור טורי.
