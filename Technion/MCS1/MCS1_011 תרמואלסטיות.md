# נגדים חשמליים

>[!Question]- האם הקיום של זרם חשמלי מעיד על מפל מתחים? 
>

אנו יודעים מ[[PHY2_006 דינמיקה מגנטית#חוק ההשראה של פאראדיי|חוק פאראדיי]] שכאשר יש לנו סליל מוליך שעובר בניצב למישורו שדה מגנטי, מתפתח כא"מ בסליל ולפיכך זרם דרכו.

אנו מכירים שנגד לינארי מקיים את חוק אוהם:
$$V=IR$$
כאשר היחידות של ההתנגדות הן:
$$[R]=\dfrac{\pu{V}}{\pu{C.s^{-1}}}=\dfrac{\pu{V^{2}}}{\pu{W}}=\pu{\Omega}$$

האנרגיה הנפלטת מהנגד (לרוב כחום) היא:
$${W}_{R}=IV=I^{2}R$$
יש לו יחידות של הספק:
$$[{W}_{R}]=\pu{W}$$
בהנחה ולנגד יש התנגדות הפרופורציונאלית לאורך הנגד ופרופורציונאלית הפוכה לשטח החתך שלו:
$$R=\dfrac{L}{A}{r}_{e}$$
כאשר ${r}_{e}$ הוא ההתנגדות הסגולית של הנגד החשמלי, עם יחידות של:
$$[{r}_{e}]=\dfrac{\pu{\Omega m^{2}}}{m}=\pu{\Omega m}$$

# נגדים תרמיים
בנגד חד-ממדי לינארי, הקשר בין השטף החשמלי לטמפרטורה נתון ע"י [[HTF1_002 הולכה חד-ממדית במצב מתמיד#משוואת ההולכה|משוואת ההולכה]]:
$$q=-k \dfrac{ \partial T }{ \partial x }$$
כאשר השטף התרמי בעל יחידות:
$$[q]=\dfrac{\pu{W}}{\pu{m^{2}}}$$
וההולכה התרמית היא:
$$[k]=\dfrac{\pu{W}}{\pu{m.K}}$$

הדיברגנץ של שטף החום הוא כמות החום המצטברת בחומר. הצטברות חום זו מגדילה את הטמפרטורה לפי:
$$\begin{gathered}
\rho C \dfrac{ \partial T }{ \partial t } =-\dfrac{ \partial q }{ \partial x }  \\[1ex]
\rho C \dfrac{ \partial T }{ \partial t } =\dfrac{ \partial  }{ \partial x } \left( k \dfrac{ \partial T }{ \partial x }  \right)
\end{gathered}$$
כאשר $\rho$ הוא הצפיפות הנמדדת ב-$\pu{kg.m^{-3}}$, ו-$C$ הוא קיבול החום הסגולי הנמדד ב- $\pu{J.kg^{-1}.K^{-1}}$.

אם מספקים עוד חום למערכת אז המשוואה השולטת היא:
$$\rho C \dfrac{ \partial T }{ \partial t } =\dfrac{ \partial  }{ \partial x } \left( k \dfrac{ \partial T }{ \partial x }  \right)+Q$$
כאשר $Q$ הוא החום המסופק למערכת ביחידות של $\pu{W.m^{-3}}$.

# משוואת החום החסרת ממדים
נביט במשוואת החום לעיל:
$$\rho C \dfrac{ \partial T }{ \partial t } =k \dfrac{ \partial ^{2}T }{ \partial x^{2} } +Q$$
ננרמל את המרחב ע"י האורך האופייני $L$ - $\tilde{x}=x/L$:
$$\dfrac{\rho C}{k}L^{2}\dfrac{ \partial T }{ \partial t } =\dfrac{ \partial ^{2}T }{ \partial \tilde{x}^{2} } +\dfrac{Q}{k}L^{2}$$
נגדיר את ה[[HTF1_002 הולכה חד-ממדית במצב מתמיד#משוואת ההולכה|דיפוסיביות התרמית]] - $\alpha^{*}=\dfrac{k}{\rho C}$
$$\dfrac{L^{2}}{\alpha ^{*}} \dfrac{ \partial T }{ \partial t } =\dfrac{ \partial ^{2}T }{ \partial \tilde{x}^{2} } +\dfrac{Q}{k}L^{2}$$
# שפעול תרמואלסטי
- Use small strains induced by temperature increase to achieve large displacements.
- Use Joule heating to increase temperature
- Use conduction and convection to decrease temperature

Advantages
- Simple structures and materials
- Compatible with microfabrication technologies
- IC voltages can produce milliwatts of heat

Disadvantages
- High power consumption
- Relatively slow

Three main thermal actuators:

- Hot/cold arms
- Bimorph actuator
- Thermal buckling actuator
